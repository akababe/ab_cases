# Interactive Features Guide

## Keyboard Navigation

Navigate between cases quickly using your keyboard:

| Shortcut | Action |
|----------|--------|
| `→` or `j` | Next case (loops to first) |
| `←` or `k` | Previous case (loops to last) |
| `/` or `Ctrl+K` | Focus search input |
| `Ctrl+D` | Toggle dark mode |
| `Shift+?` | Show keyboard shortcuts help |

**Note**: Keyboard shortcuts work when the sidebar is closed. If the sidebar is open, use the search input instead.

## Search & Filter

### Desktop
- Type in the search box in the **sidebar** to filter cases
- Press `/` or `Ctrl+K` to quickly focus the search box

### Mobile
- Tap the search input in the **top bar**
- Press `/` or `Ctrl+K` to focus it
- Search filters cases in real-time as you type

**Tip**: If no cases match your search, you'll see "No cases found"

## Dark Mode

Toggle between light and dark themes:

- **Click** the sun/moon icon (☀️/🌙) in the top bar
- **Keyboard**: Press `Ctrl+D`

Your theme preference is saved automatically and will be remembered next time you visit.

## Reading Progress

A thin gradient bar appears at the top of the page showing your scroll progress through the current case. This is especially useful for longer cases.

## Page Transitions

When switching between cases:
- Content fades in smoothly (350ms)
- Sections slide up with staggered timing for visual hierarchy
- You're automatically scrolled to the top of the new case

## Animations & Interactions

- **Hover buttons**: Buttons lift slightly on hover and click with tactile feedback
- **Active indicators**: The current case in the sidebar shows an arrow (→)
- **Focus outlines**: When tabbing through content, interactive elements get clear focus indicators (3px accent outline)

## Accessibility

The entire interface is keyboard accessible:

- **Tab** to move between interactive elements
- **Enter** or **Space** to activate buttons and select cases
- **Arrow keys** to navigate cases (see shortcuts above)
- All buttons have ARIA labels
- Empty search states show clear messaging

## Mobile Experience

On smaller screens:

- The sidebar converts to a slide-out drawer
- A dedicated top bar shows the current case title and controls
- Search is available directly in the top bar
- The menu button (≡) toggles the sidebar
- Keyboard shortcuts still work for efficient navigation

## Tips & Tricks

1. **Quick search**: Press `/` to search immediately without clicking
2. **Power navigation**: Use `j`/`k` if you prefer Vi keybindings
3. **Dark mode preference**: System theme is detected automatically; manually toggle with `Ctrl+D` to override
4. **Focus visible**: Tab through the interface to see clearly marked focus states for keyboard navigation
5. **Progress tracking**: Use the reading progress bar to gauge how much of a case remains
